# 🃏 Telegram Tarot Bot – Daily Card, Reroll, Reversed Cards

Telegram bot for Tarot readings. Daily card, reroll, reversed cards.

## 🚀 Demo

[Link to the bot](@MyTarotBotMyTarotBot)

## 📦 Tech Stack

- Python 3.10+
- aiogram 3.x
- SQLite
- aiosqlite

## 🏗️ Architecture

- `crud/` — database interactions
- `services/` — business logic
- `handlers/` — command handlers
- `keyboards/` — keyboards layout

## 🚀 Run Locally

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Create `.env` file from `.env.example`
4. Run: `python src/main.py`
