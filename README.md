# 🔮 Tarot Magerie Bot
# 🎮 Живой бот: [@MyTarotBotMyTarotBot](https://t.me/MyTarotBotMyTarotBot)

Телеграм-бот для гадания на Таро. Карта дня, перевёрнутые карты, случайная рубашка.

![Image](https://github.com/user-attachments/assets/248952eb-f1be-41af-aa92-014a3e3a647a)

## 🃏 Особенности

- **78 карт Таро** с прямыми и перевёрнутыми толкованиями (156 значений)
- **Случайная рубашка** из коллекции (админка для загрузки/удаления)
- **Одноразовый реролл** — кнопка исчезает после использования
- **Карта дня** — одна карта в сутки (сброс в полночь)
- **Админ-панель** — загрузка рубашек, получение file_id
- **PostgreSQL** — готова к масштабированию
- **Тесты** — 27 тестов (pytest, покрытие CRUD, сервисов, хэндлеров)
- **Персонализированный AI-Оракул** с учётом контекста пользователя.

## 🚀 Запуск

### Локально (PostgreSQL)

1. **Установить PostgreSQL** и создать БД:
   sql
   CREATE DATABASE tarot_bot;
   CREATE USER bot_user WITH PASSWORD 'bot_pass';
   GRANT ALL PRIVILEGES ON DATABASE tarot_bot TO bot_user;
   
2. **Клонировать репозиторий:**
    bash 
    git clone https://github.com/mageri9/TAROT_magerie_BOT.git
    cd tarot-bot

3. **Установить зависимости:**
    bash
    pip install -r requirements.txt
   (для тестов и скриптов)
    pip install -r requirements-dev.txt
   
4. **Создать .env по примеру .env.example**

5. **Запустить бота**

### 📁 Структура проекта

- **`src/`** — основной код
  - `core/` — ядро (config, db, redis, router)
  - `crud/` — слой работы с базой данных
  - `services/` — бизнес-логика
  - `handlers/` — обработчики команд и колбэков
  - `keyboards/` — клавиатуры
  - `filters/` — кастомные фильтры (`IsAdmin`, `ChatType`)
  - `middlewares/` — логирование и защита (Rate Limiter)
  - `database/` — инициализация и миграции (Alembic)

- **`scripts/`** — вспомогательные скрипты
  - `upload_deck.py` — загрузка колоды из 78 карт
  - `migrate_cards.py` — миграция данных из SQLite
  - `backup.sh` — скрипт для автоматического бэкапа БД

- **`tests/`** — тесты (pytest)
  - `crud/` — тесты CRUD
  - `handlers/` — тесты хендлеров
  - `services/` — тесты сервисов
  - `conftest.py` — фикстуры

## 🛠️ DevOps & Инфраструктура
- **Полная контейнеризация:** Docker Compose (бот, PostgreSQL, Redis).
- **CI/CD:** Автоматический деплой на VPS через GitHub Actions.
- **Мониторинг:** Админ-панель с метриками (CPU/RAM, аптайм, статусы сервисов).
- **Надёжность:** Rate Limiter на Redis, защита от спама.
- **Бэкапы:** Автоматические (cron) и ручные (по кнопке) дампы БД с ротацией.
- **Логирование:** Структурированные логи с ротацией (Loguru).

## 🤖 AI-Оракул
- **Персонализированные предсказания** с учётом контекста пользователя.
- **RAG-логика**: AI опирается на базу из 156 уникальных толкований.
- **Стек AI**: Gemma 4 (26B) через AITUNNEL API.
- **Экономичность**: одно предсказание стоит ~0.005 ₽.
