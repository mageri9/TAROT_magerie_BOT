# 🃏 Телеграм-бот для гадания на Таро. Карта дня, реролл, перевёрнутые карты.

Телеграм-бот для гадания на Таро. Карта дня, реролл, перевёрнутые карты.

## 🚀 Демо

[Ссылка на бота](@MyTarotBotMyTarotBot)

## 📦 Стек

- Python 3.10+
- aiogram 3.x
- SQLite
- aiosqlite

## 🏗️ Архитектура

- `crud/` — работа с БД
- `services/` — бизнес-логика
- `handlers/` — обработчики команд
- `keyboards/` — клавиатуры

## 🚀 Запуск локально

1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Создать `.env` по примеру `.env.example`
4. Запустить: `python src/main.py`
